Synthèse vocale

Utilisée par les boutons 🔊 (lire à voix haute). La disponibilité dépend du navigateur/OS.
⚠ Note importante : La reconnaissance vocale de cette appli utilise l'API native du navigateur (webkitSpeechRecognition / SpeechRecognition). Dans Chrome et la plupart des navigateurs Chromium, cette API envoie l'audio vers les serveurs de Google pour traitement. Sur Firefox Desktop, l'API peut être absente ou utiliser un service tiers selon la version. Si vous avez besoin d'une reconnaissance strictement locale, consultez la feuille de route LibreA11y (Vosk, Whisper…).
