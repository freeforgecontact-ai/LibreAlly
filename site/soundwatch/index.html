SoundWatch — Alertes visuelles et vibrantes pour sons importants

ℹ️ Ce que cette app fait vraiment

Distinguer finement les sons (sonnette vs alarme vs bébé) nécessite un modèle de classification audio entraîné — non disponible localement dans un navigateur. Cette app détecte le niveau sonore et les patterns d'amplitude via l'API Web Audio du navigateur, et déclenche des alertes visuelles et vibrantes paramétrables. Les boutons « Tester » simulent chaque type d'alerte. La fonction « Apprendre » calibre un seuil personnalisé basé sur le niveau moyen d'un son.

🎤 Écoute en direct

Utilise le microphone pour mesurer le niveau sonore ambiant. Une alerte se déclenche si le niveau dépasse le seuil configuré.
