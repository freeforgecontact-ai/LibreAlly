SoundWatch — Alertes visuelles et vibrantes pour sons importants

ℹ️ Ce que cette démo fait vraiment

Distinguer les sons (sonnette vs alarme vs bébé qui pleure) nécessite un modèle de classification audio entraîné — non disponible en local dans un navigateur. Cette démo détecte le niveau sonore (sons forts, claps) via l'API Web Audio du navigateur, et illustre le système d'alertes visuelles et vibrantes. Les boutons de test simulent chaque type d'alerte pour montrer l'expérience complète.
