2 — Clic par maintien (dwell)
Maintenez le pointeur sur une cible sans cliquer. Elle s'active automatiquement.
Problème résolu : Les tremblements provoquent des clics involontaires ou manqués. Avec le dwell, on maintient simplement le curseur immobile — aucun clic nécessaire.

3 — Anti-double-déclenchement
Un délai minimal ignore les appuis répétés involontaires (spasmes, rebonds).
Problème résolu : Un tremblement peut provoquer plusieurs clics en rafale sur le même bouton. Le filtre anti-rebond bloque tout déclenchement survenant trop vite après le précédent. Appuyez rapidement plusieurs fois pour voir les « appuis ignorés ».
