🆘 Aide — Comment utiliser SwitchBoard

SwitchBoard permet de piloter des actions à l'écran avec un ou deux contacteurs physiques (boutons adaptés).

  • Balayage automatique : le surlignage avance seul de case en case. Appuyez sur votre contacteur (ou Espace) pour sélectionner la case en surbrillance.
  • Par étapes : appuyez sur Entrée (ou le bouton Avancer) pour avancer manuellement, puis Espace (ou Sélectionner) pour valider.
  • Mode ligne-colonne : activez-le pour d'abord sélectionner une ligne entière, puis une colonne dans cette ligne.
  • Temps d'acceptation : délai avant qu'une pression soit prise en compte (évite les faux-contacts).

Les boutons à l'écran (Sélectionner / Avancer) remplacent les contacteurs physiques pour la démonstration.
