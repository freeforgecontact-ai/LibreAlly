Aide — SwitchBoard

SwitchBoard permet de piloter des actions avec un ou deux contacteurs physiques.

  • Balayage auto : le surlignage avance seul. Appuyez sur Espace (ou le bouton Selectionner) pour choisir.
  • Pas-a-pas : appuyez sur Entree pour avancer, puis Espace pour selectionner.
  • Ligne-col : selectionnez d'abord une ligne, puis une cellule dans cette ligne.
  • Groupes : les cellules sont regroupees ; selectionnez un groupe puis une cellule.
  • 2 contacteurs : Espace = selectionner, Entree = avancer independamment.
  • Temps d'acceptation : delai avant qu'une pression soit prise en compte.
  • Mode edition : activez la bascule "Editer" pour modifier les cellules.
  • Profils : sauvegardez vos reglages (vitesse, mode, etc.) sous un nom.

Les boutons Selectionner et Avancer remplacent les contacteurs physiques.
