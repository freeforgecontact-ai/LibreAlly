📖 VoxRead — démo

démo · 100% hors-ligne · aucune donnée envoyée

⚠️ Limites de cette démo

Cette démo mono-fichier couvre la lecture vocale de texte saisi et de fichiers .txt. Elle fonctionne entièrement hors-ligne, sans serveur, sans données envoyées.

  • PDF / images / scan papier : nécessitent un moteur OCR embarqué (ex : Tesseract.js) — non inclus dans cette démo, disponible en production.
  • Lecture par caméra (« lire le monde ») : requiert OCR en temps réel — voir onglet Caméra pour les détails.

Importer un fichier texte

Caméra — Lire le monde

Pourquoi ce bouton est désactivé ?

La lecture par caméra (OCR en temps réel) est la fonctionnalité phare de VoxRead en production. Elle permet de pointer la caméra vers n'importe quel texte — étiquette, panneau, livre imprimé — et de l'entendre aussitôt.

  • Elle nécessite un moteur de reconnaissance de caractères (OCR) embarqué, typiquement Tesseract.js (~7 Mo de données de modèle).
  • Ce fichier de démo est volontairement 100% autonome et hors-ligne, sans aucune ressource externe, ce qui rend l'intégration d'un modèle OCR incompatible avec la contrainte d'un seul fichier HTML.
  • En production, la fonctionnalité est entièrement locale : aucune image n'est envoyée sur un serveur.

→ Cette démo couvre la lecture de texte saisi et de fichiers .txt — fonctionnalités pleinement opérationnelles ci-dessus.

Raccourcis clavier

L'application est entièrement pilotable au clavier. Les raccourcis fonctionnent depuis n'importe quel onglet.

  • Alt + L Lire le texte / fichier actif
  • Alt + P Pause / Reprendre
  • Alt + S Stop
  • Alt + E Épeler le mot
  • Alt + 1 Aller à l'onglet Texte
  • Alt + 2 Aller à l'onglet Fichier
  • Alt + 3 Aller à l'onglet Caméra
  • Alt + 4 Aller aux Raccourcis
  • Flèches ←→ Naviguer entre les onglets
  • Échap Stop si lecture en cours
  • Tab Passer d'un contrôle au suivant
  • Entrée / Espace Activer un bouton

Navigation sans voir

VoxRead est conçu pour être utilisé sans regarder l'écran :
toutes les actions sont vocalisées (annonces aria-live), les boutons ont des libellés aria-label explicites, le focus est toujours visible (anneau de 3 px couleur or), et les cibles font au minimum 56 px. Un lecteur d'écran tel que NVDA, JAWS ou VoiceOver fonctionne pleinement avec cette interface.
