        Raccourcis clavier

        L'application est entierement pilotable au clavier.

        • Alt + L Lire le texte / fichier actif
        • Alt + P Pause / Reprendre
        • Alt + S Stop
        • Alt + E Epeler le mot
        • Alt + B Ajouter un marque-page
        • Alt + 1 Onglet Texte
        • Alt + 2 Onglet Fichier
        • Alt + 3 Onglet OCR Image
        • Alt + 4 Onglet OCR Camera
        • Alt + 5 Onglet Bibliotheque
        • Alt + 6 Onglet Reglages
        • Alt + 7 Onglet Raccourcis
        • Fleches gauche/droite Naviguer entre onglets
        • Echap Stop si lecture en cours
        • Tab Passer d'un controle au suivant
        • Entree / Espace Activer un bouton

        Navigation sans voir

        VoxRead est concu pour etre utilise sans regarder l'ecran : toutes les actions sont vocalisees (annonces aria-live), les boutons ont des libelles aria-label explicites, le focus est toujours visible (anneau de 3 px couleur or), et les cibles font au minimum 44-56 px. Un lecteur d'ecran tel que NVDA, JAWS ou VoiceOver fonctionne pleinement.

        OCR sur l'appareil : le moteur Tesseract.js LSTM traite l'image localement. Le modele francais (~6 Mo) se telecharge au 1er usage (puis mis en cache), ou peut etre embarque pour un fonctionnement 100% hors-ligne.
